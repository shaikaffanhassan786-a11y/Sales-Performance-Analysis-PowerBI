# Sales-Performance-Analysis-PowerBI
A Power BI dashboard project analyzing sales performance, profit trends, discounts, and regional insights using interactive visuals and DAX. Built to demonstrate real-world data analysis skills.
# Project Overview

This project focuses on analyzing sales performance data using Power BI to uncover meaningful business insights.
The dashboard helps in understanding sales trends, profit behavior, discount impact, and regional performance through interactive visuals.

The goal of this project is to demonstrate practical data analysis, visualization, and business intelligence skills using Power BI.
# Objectives

Analyze overall sales and profit performance

Identify top and bottom performing products

Understand sales trends over time

Analyze discount impact on profit

Compare performance across cities

Create an interactive dashboard for business decision-making
## Tools & Technologies Used

Power BI

Microsoft Excel / CSV Dataset

# Key Insights
A small group of products contributes to a major portion of total revenue.

Higher discounts often lead to reduced profit margins.

Sales show seasonal trends across different time periods.

Some cities generate high sales but lower profit margins.

Profit does not always increase with higher sales due to discounting.
